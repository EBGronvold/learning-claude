# eirikgronvold.com

Personal portfolio site — static HTML, no build tools.

## Preview locally

Open `index.html` in your browser.

## Deploy to Vercel

```bash
# From this directory:
npx vercel

# Follow the prompts, then connect your domain in the Vercel dashboard:
# Settings → Domains → Add eirikgronvold.com
```

## Customization

All design tokens are in the `:root` CSS variables at the top of `index.html`.
Edit the HTML directly to update content — no build step needed.
