# Learning Roadmap — Claude Code to Career

**Goal:** Financial and geographic independence within 3-6 months by mastering
AI-powered development and selling those skills to businesses.

**Strategy:** Learn by building. Every week produces something real.

---

## Phase 1: Foundations (Weeks 1-2)
> Get comfortable with the tools. Everything else depends on this.

### Week 1 — Claude Code + Git
- [ ] Complete Anthropic Skilljar course (anthropic.skilljar.com)
- [ ] Learn Claude Code core workflow: ask, edit, review, commit
- [ ] Understand CLAUDE.md and how it shapes Claude's behavior
- [ ] Git basics: init, add, commit, push, pull, branch, merge
- [ ] Push this learning-claude repo to GitHub
- [ ] Practice: Have Claude build a simple script, review the code, understand it

### Week 2 — Terminal + Development Environment
- [ ] Essential terminal commands (cd, ls, mkdir, rm, mv, cp, cat, grep)
- [ ] npm basics: init, install, run scripts
- [ ] Create your first Node.js project with Claude's help
- [ ] Learn how Claude Code uses tools (Read, Write, Edit, Bash, Grep, Glob)
- [ ] Practice: Build a CLI tool that does something useful (file organizer, etc.)

**Milestone:** You can navigate any project, use Claude Code fluently, and push code to GitHub.

---

## Phase 2: APIs — How Software Talks (Weeks 3-4)
> APIs are the pipes that connect everything. Master this and you can build anything.

### Week 3 — API Fundamentals
- [ ] What is an API? (REST, HTTP methods, JSON, status codes)
- [ ] Use the Claude/Anthropic API directly (get your API key)
- [ ] Make API calls with fetch/axios in Node.js
- [ ] Build a simple app that talks to Claude's API
- [ ] Understand API keys, authentication, rate limits

### Week 4 — Building Integrations
- [ ] Connect to a third-party API (weather, news, or a business tool)
- [ ] Build a simple Express.js API server
- [ ] Learn about webhooks (how services notify each other)
- [ ] Practice: Build a tool that combines 2+ APIs (e.g., "summarize today's news using Claude")

**Milestone:** You can call any API, build basic API servers, and connect services together.

---

## Phase 3: MCPs — Claude's Superpowers (Weeks 5-6)
> MCP lets Claude access real-world data and tools. This is what makes you dangerous.

### Week 5 — Understanding and Using MCPs
- [ ] What is MCP (Model Context Protocol) and why it matters
- [ ] Install and configure existing MCP servers
- [ ] Connect Claude to a database via MCP
- [ ] Connect Claude to file systems, web search, or other tools
- [ ] Understand the MCP architecture: hosts, servers, tools, resources

### Week 6 — Building Custom MCPs
- [ ] Build your first custom MCP server
- [ ] Create an MCP that connects to a business API (CRM, spreadsheet, etc.)
- [ ] Practice: Build an MCP that solves a real business problem

**Milestone:** You can extend Claude with any data source or tool. This is a rare, high-value skill.

---

## Phase 4: Web Development (Weeks 7-9)
> Build things people can see and use. This is how you deliver value to clients.

### Week 7 — HTML, CSS, JavaScript Basics
- [ ] Core HTML structure and semantic elements
- [ ] CSS styling, flexbox, responsive design
- [ ] JavaScript fundamentals (variables, functions, DOM, events)
- [ ] Build a simple landing page with Claude's help

### Week 8 — Modern Web Apps
- [ ] React or Next.js fundamentals
- [ ] Components, state, props, routing
- [ ] Build a functional web app (dashboard, tool, or calculator)
- [ ] Connect your app to an API

### Week 9 — Deployment
- [ ] Deploy to Vercel or Netlify (free)
- [ ] Custom domains and DNS basics
- [ ] Environment variables in production
- [ ] Practice: Deploy a complete project that's live on the internet

**Milestone:** You can build and deploy professional web applications.

---

## Phase 5: Automation + AI Agents (Weeks 10-12)
> This is where you start making money. Automate business processes.

### Week 10 — Scripting and Automation
- [ ] Build scripts that automate repetitive tasks
- [ ] Cron jobs and scheduled tasks
- [ ] Web scraping basics (ethical, with permission)
- [ ] Email automation, file processing, data transformation

### Week 11 — AI Agents
- [ ] What are AI agents and how they work
- [ ] Build an agent that completes multi-step tasks
- [ ] Claude Agent SDK fundamentals
- [ ] Tool use — giving agents the ability to take actions

### Week 12 — Workflow Automation
- [ ] n8n or Make for visual workflow building
- [ ] Connecting everything: API + MCP + Agent + UI
- [ ] Build a complete automation for a realistic business use case
- [ ] Document it as a case study

**Milestone:** You can build AI-powered automations that save businesses hours per week.

---

## Phase 6: Go to Market (Weeks 13-16)
> Package everything and start earning.

### Weeks 13-14 — Portfolio + Positioning
- [ ] Build your portfolio website (showcasing 3-5 best projects)
- [ ] Write case studies for each project (problem → solution → result)
- [ ] Create your GitHub profile README
- [ ] Define your service offering (what you do, for whom, at what price)
- [ ] Choose your niche: AI agency, automation consulting, SaaS, or hybrid

### Weeks 15-16 — First Clients
- [ ] Outreach strategy: LinkedIn, cold email, communities
- [ ] Build a lead magnet (free tool, audit, or guide)
- [ ] Price your services (start with value-based pricing)
- [ ] Land your first client or build your first revenue-generating project
- [ ] Deliver, get a testimonial, iterate

**Milestone:** You have a portfolio, a clear offering, and your first paying client.

---

## Key Principles

1. **Build every day.** Even 30 minutes of building beats 3 hours of tutorials.
2. **Use Claude Code for everything.** The more you use it, the faster you get.
3. **Ship imperfect work.** A deployed ugly app beats a perfect app in your head.
4. **Document as you go.** Your learning becomes content, content attracts clients.
5. **Think in business problems.** Every skill maps to a problem businesses will pay to solve.
